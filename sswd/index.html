9. OverPy 代码结构

# 全局变量声明 globalvar gameStatus # 0=等待 1=分配 2=描述 3=投票 4=结算 globalvar civilianWord # 平民词语 globalvar undercoverWord # 卧底词语 globalvar currentSpeaker # 当前描述玩家 globalvar roundNumber # 当前回合 playervar playerWord # 每个玩家的词语 playervar isUndercover # 是否卧底 playervar hasVoted # 是否已投票 playervar voteTarget # 投票目标 rule "游戏初始化": @Event eachPlayer @Condition gameStatus == 0 initializeGame() rule "分配词语": @Condition gameStatus == 1 assignWords() showWordToPlayer() rule "描述阶段": @Condition gameStatus == 2 updateSpeaker() showTimer() rule "投票阶段": @Condition gameStatus == 3 collectVotes() announceResult()

10. 卧底数量自动配置

4-5 玩家
1 卧底
6-8 玩家
2 卧底
9-12 玩家
3 卧底
13+ 玩家
4 卧底
# 卧底数量计算公式 macro getUndercoverCount(playerCount): if playerCount >= 13: return 4 elif playerCount >= 9: return 3 elif playerCount >= 6: return 2 else: return 1
