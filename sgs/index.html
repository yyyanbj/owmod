📥 如何导入游戏

  1. 编译项目：overpy compile src/main.opy --output dist/sanguosha.ws
  2. 打开 Overwatch，选择 自定义游戏
  3. 进入 创意工坊 (Workshop)
  4. 点击 导入脚本 (Import Script)
  5. 选择 dist/sanguosha.ws 文件
  6. 创建房间，邀请 2-10 名玩家
  7. 主持人按住互动键开始游戏

📏 距离系统

距离计算基于座位差，并受以下因素影响：

  • 基础距离 = |座位 A - 座位 B|
  • 卢西奥【加速】：计算距离时 -1
  • 温斯顿【跳跃背包】：计算距离时 -1
  • 托比昂【炮台】：别人计算与他的距离 +1
攻击范围：默认攻击范围为 1。需要距离 ≤ 攻击范围才能使用杀。

📋 游戏流程

  1. WAITING - 等待阶段 - 等待玩家加入（2-10 人）
  2. PREPARING - 准备阶段 - 主持人按住互动键开始，分发身份
  3. SELECTING_HEROES - 英雄选择 - 按 F1-F12 选择 OW 英雄
  4. PLAYING - 游戏进行中
    • 判定阶段 - 延时锦囊生效
    • 摸牌阶段 - 摸 2 张牌
    • 出牌阶段 - 使用任意张牌（杀受次数限制）
    • 弃牌阶段 - 手牌数 ≤ 当前体力
    • 结束阶段 - 轮到下家
  5. GAME_OVER - 游戏结束 - 显示胜利信息，10 秒后重置

⚙️ 技术实现细节

项目架构：
• OverPy 编译器：https://github.com/Zezombye/overpy
• 源代码：src/*.opy
• 编译输出：dist/sanguosha.ws
• 设计文档：docs/superpowers/specs/
• 实现计划：docs/superpowers/plans/

模块结构

  • main.opy - 主入口，游戏主循环
  • config.opy - 配置、枚举、常量
  • identity.opy - 身份系统、胜利条件
  • hero_selection.opy - 英雄选择系统
  • heroes/*.opy - 12 个英雄技能实现
  • card.opy - 卡牌系统、距离计算
  • turn.opy - 回合流程管理
  • damage.opy - 伤害计算、技能触发
  • hud.opy - HUD 显示系统

OverPy 语法

#!include "config.opy" globalvar gameStatus playervar currentHP enum GameStatus: WAITING = 0 PLAYING = 1 rule "游戏开始": @Condition gameStatus == GameStatus.WAITING @Condition hostPlayerVar.isHoldingButton(Button.INTERACT) gameStatus = GameStatus.PLAYING
